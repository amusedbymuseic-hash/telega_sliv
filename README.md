# Угадай подружку

Статический сайт для GitLab Pages.

## Публикация
1. Создай пустой проект в GitLab.
2. Загрузи все файлы из этой папки в корень ветки `main`.
3. Открой **Deploy → Pages** после успешного pipeline.
